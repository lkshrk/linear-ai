# Example Feature Issue

```markdown
# Show recent workflow runs on the project overview

## Problem / Opportunity

Operators need to see whether recent workflow runs succeeded without opening each workflow individually.

## Desired Outcome

The project overview shows recent workflow runs and their status so an operator can spot failures quickly.

## User / Actor

Internal operator reviewing project health.

## Current Behavior

The overview page shows project metadata but not recent workflow execution status.

## Proposed Behavior

Add a recent workflow runs section to the project overview with status, run time, and link to details.

## Context

- Product area: project overview
- Related workflow: workflow execution monitoring
- Related repos: web, backend
- Constraints: keep overview scannable
- Non-goals: full workflow run history page

## Evidence / Links

- Related operator note: linked in Linear
```

Recommended labels:

- `feature`
- `llm-refine`
